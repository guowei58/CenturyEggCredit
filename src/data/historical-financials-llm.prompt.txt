ABSOLUTE RULES

1. Original numbers only
Use only company-reported figures from original sources.

2. No invention
Do not invent, estimate, interpolate, smooth, or infer undisclosed values.

3. No silent blanks
Every missing field must be handled only as one of:
- Directly reported
- Correctly derived from reported figures
- Not separately disclosed

4. “Not separately disclosed” is limited
“Not separately disclosed” may be used only for a specific line item within an otherwise processed statement.

It may not be used for:
- an entire missing year
- an entire missing quarter
- an entire unprocessed statement
- an entire unprocessed filing

If a filing exists and contains the relevant period, you must process it.

5. Preserve management presentation
Preserve management’s:
- row order
- labels

6. No careless flattening
If presentation changes over time:
- use the latest statement format as the master template,
- map older lines only when clearly equivalent,
- otherwise preserve separate legacy rows with clear labels.

7. Full period processing required
A period is not complete unless the relevant original filing has been examined.

8. Full statement processing required
A statement is not complete unless every row management presents has been addressed.

CRITICAL QUARTERLY RULES

1. Quarterly Income Statement
Use directly reported quarterly figures from 10-Q where available.

For Q4, if standalone Q4 is not directly presented, derive:

Q4 = FY – 9M

Every such value must be clearly labeled:

Derived from FY less 9M

Do not label derived Q4 figures as directly reported.

2. Quarterly Cash Flow Statement
Quarterly cash flow must be standalone quarter, not YTD.

Because 10-Q cash flow is usually YTD:

- Q1 = Q1 YTD
- Q2 = Q2 YTD – Q1 YTD
- Q3 = Q3 YTD – Q2 YTD
- Q4 = FY – 9M

Every derived quarterly cash flow value must be clearly labeled as derived.

3. Quarterly Balance Sheet
Use actual reported quarter-end balance sheet figures directly from 10-Q / 10-K.

Do not derive balance sheet amounts unless absolutely necessary and logically unavoidable.
